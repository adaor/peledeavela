# 🌰 Movimento Pele de Avelã

Bem-vindo ao repositório oficial do **Movimento Pele de Avelã**. Este é um espaço dedicado ao autocuidado, à valorização da identidade e ao bem-estar.

## 🚀 Sobre o Projeto
Este site foi desenvolvido para ser a vitrine digital do movimento, apresentando uma estética clean, sofisticada e minimalista. O foco é proporcionar uma experiência de navegação leve e direta para os usuários.

## 🛠️ Tecnologias Utilizadas
* **HTML5**: Estrutura semântica do site.
* **CSS3**: Estilização personalizada, com foco em design responsivo (adaptável para celulares).
* **GitHub Pages**: Hospedagem gratuita e segura.
